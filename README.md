## 주요 기능

- 카테고리별 레시피 표시
- 레시피 검색 기능
- 페이지네이션
- 랜덤 메뉴를 정할 수 있는 룰렛 게임
- 레시피 영양 정보, 조리 방법 제공
- 레시피 링크 공유
